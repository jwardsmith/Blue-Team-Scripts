- Contains a compressed RAM Image
- %SystemDrive%\hiberfil.sys

Page and Swap Files
- %SystemDrive%\pagefile.sys
- %SystemDrive%\swapfile.sys (Win8+\2012+)

Memory Dump
- %WINDIR%\MEMORY.DMP
